Projects

CREO for Retail is a state-of-the-art automated design system built to generate localized, brand-compliant retail marketing assets at scale. Unlike standard generative tools, it operates on a "Compliance-First" architecture that strictly enforces brand guidelines, legal requirements, and layout safety zones before generating any content. The system uses a decoupled React and FastAPI architecture, incorporating computer vision to detect human presence and a geometric layout solver to prevent overlaps, ensuring creatives are automatically built for various formats without breaking rules.
SafeMint is a trust-first token creation and liquidity-locking platform built on the QIE Mainnet. Designed to protect users from rug pulls and malicious token launches, the platform enforces fair launches, locked liquidity, and visible trust signals from day one. By ensuring 100% of provided liquidity is locked automatically for 6–12 months, SafeMint allows users to instantly verify token safety before purchasing, offering a robust solution to secure the decentralized ecosystem.
IPFS CDN is a decentralized content delivery system built on IPFS, exploring web infrastructure that functions independently of centralized cloud hosting. It provides resilient, peer-to-peer file sharing and static site hosting. The project was featured in several discussions about the decentralized web, see the DWeb Conference talk and this technical blog post. Also related to this, check out my earlier distributed experiments like p2p-chat and distributed-ledger-demo.
BaseLine is a lightweight utility project demonstrating core software engineering principles and streamlined setup.
